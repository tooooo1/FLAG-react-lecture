## 1. 브랜치 생성

총 2가지 브랜치를 생성합니다.

- `1/username_working` : 과제를 진행할 브랜치 (앞에 숫자는 몇주차인지 적어주세요)
- `1/username` : 과제를 제출할 브랜치 (앞에 숫자는 몇주차인지 적어주세요)

## 2. PR
```
base : 1/username ← compare : 1/username_working
```

- 본인 branch로 제출되는지 확인해 주세요


## 3. LABEL
